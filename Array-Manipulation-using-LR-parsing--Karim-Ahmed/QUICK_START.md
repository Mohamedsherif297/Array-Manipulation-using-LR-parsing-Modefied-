# Quick Start Guide

## 🚀 Running the Compiler

### Option 1: Integrated Pipeline (Recommended for Testing)
```bash
# Compile
g++ -std=c++17 Main.cpp -o Main

# Run (processes hardcoded test string)
./Main

# Output: ast.json
```

### Option 2: Full Pipeline (All 4 Phases)

#### Step 1: Lexing + Parsing
```bash
./Main
# Output: ast.json
```

#### Step 2: Semantic Analysis
```bash
cd semantic
g++ -std=c++17 semantic_main.cpp ast_builder.cpp semantic_analyzer.cpp \
    semantic_output.cpp symbol_table.cpp -o semantic_main
./semantic_main ../ast.json .
# Output: annotated_ast.json, symbol_table.json
```

#### Step 3: Code Generation
```bash
cd ../codegen
make
./codegen ../semantic/annotated_ast.json ../semantic/symbol_table.json ir.txt
# Output: ir.txt (Three-Address Code)
```

## 📁 Project Structure

```
lexer/      → Tokenization
parser/     → LR(1) Parsing + AST
semantic/   → Type Checking
codegen/    → TAC Generation
Main.cpp    → Entry Point
```

## 🔍 Example Input/Output

### Input (in Main.cpp)
```cpp
string src = "int x [2][2]= {{1,2},{3,4}};";
```

### Output: ast.json
```json
{
  "type": "Program",
  "children": [
    {
      "type": "DeclAssign",
      "children": [...]
    }
  ]
}
```

### Output: ir.txt
```
// DECL x  type=int
t1 = 0 * 4
x[t1] = 1
t2 = 1 * 4
x[t2] = 2
...
```

## 🛠️ Modifying the Input

Edit `Main.cpp` line 13:
```cpp
string src = "int x [2][2]= {{1,2},{3,4}};";
```

Change to your desired input, then recompile and run.

## ✅ Supported Syntax

```cpp
// Variable declarations
int x;
float y;

// Array declarations
int arr[5];
int matrix[2][3];

// Assignments
x = 5;
arr[0] = 10;

// Declaration with initialization
int x = 5;
int arr[2] = {1, 2};
int m[2][2] = {{1,2},{3,4}};

// Expressions
x = a + b * c;
y = arr[i] + 5;
```

## 🐛 Troubleshooting

### Compilation Error
```bash
# Make sure you're in the right directory
cd Array-Manipulation-using-LR-parsing--Karim-Ahmed

# Use C++17
g++ -std=c++17 Main.cpp -o Main
```

### Missing Output Files
- `ast.json` - Generated by Main.cpp
- `annotated_ast.json` - Generated by semantic_main
- `symbol_table.json` - Generated by semantic_main
- `ir.txt` - Generated by codegen

Check the grammar in `CFG.md` for supported syntax.
