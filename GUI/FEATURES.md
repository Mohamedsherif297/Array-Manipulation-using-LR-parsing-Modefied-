# GUI Features

## 🎨 Code Editor

### Monaco Editor Integration
- **Syntax Highlighting**: Full C/C++ syntax highlighting
- **Line Numbers**: Easy code navigation
- **Auto-indentation**: Smart code formatting
- **Word Wrap**: Comfortable reading of long lines
- **Dark Theme**: VS Code-inspired dark theme

### Editor Features
- Real-time editing
- Undo/Redo support
- Find and Replace (Ctrl/Cmd + F)
- Multi-cursor editing (Alt + Click)
- Code folding
- Bracket matching

## 🌳 Abstract Syntax Tree (AST) Visualization

### Tree View
- **Interactive Nodes**: Click to expand/collapse
- **Color-coded Elements**:
  - 🟢 Node types (green)
  - 🟠 Values (orange)
  - 🔵 Data types (blue)
  - 🟡 Semantic info (yellow)
- **Hierarchical Display**: Clear parent-child relationships
- **Auto-expand**: First 2 levels expanded by default

### JSON View
- Raw JSON representation
- Syntax-highlighted
- Scrollable for large trees
- Copy-friendly format

## 📊 Symbol Table

### Table View
- **Columns**:
  - Symbol name (identifier)
  - Data type (int, float, etc.)
  - Array status (Yes/No badge)
  - Dimensions (for arrays)
- **Interactive**:
  - Hover effects
  - Sortable columns
  - Sticky header

### Details View
- Complete JSON representation
- All symbol attributes
- Easy to copy and share

## ⚙️ Three-Address Code (TAC)

### Formatted Display
- **Line Numbers**: Easy reference
- **Syntax Highlighting**:
  - Comments in green
  - Variables in cyan
  - Operators in white
  - Expressions in orange
- **Statistics**: Total instruction count
- **Hover Effects**: Highlight current line

### Raw Output
- Unformatted TAC
- Exactly as generated by compiler
- Useful for debugging

## 🚀 Compilation

### Real-time Compilation
- Click "Compile & Run" button
- Progress indicator during compilation
- Fast response times

### Error Handling
- Clear error messages
- Syntax error reporting
- Semantic error display
- Network error handling

### Multi-phase Pipeline
1. **Lexical Analysis**: Tokenization
2. **Syntax Analysis**: LR(1) parsing
3. **Semantic Analysis**: Type checking
4. **Code Generation**: TAC generation

## 🎯 User Experience

### Responsive Design
- Resizable panels
- Scrollable content areas
- Adaptive layouts
- Mobile-friendly (basic support)

### Visual Feedback
- Loading spinners
- Success/error states
- Hover effects
- Smooth transitions

### Keyboard Shortcuts
- **Ctrl/Cmd + S**: Compile (when editor focused)
- **Ctrl/Cmd + F**: Find in editor
- **Ctrl/Cmd + H**: Replace in editor
- **Tab**: Indent
- **Shift + Tab**: Outdent

## 🔧 Technical Features

### Performance
- Lazy loading of components
- Efficient re-rendering
- Optimized bundle size
- Fast compilation pipeline

- Vite for fast builds
- Express backend API
- RESTful API design

- POST `/api/compile`: Compile source code
- JSON request/response
